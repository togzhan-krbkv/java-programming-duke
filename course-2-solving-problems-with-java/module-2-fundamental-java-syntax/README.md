# Module 2: Fundamental Java Syntax and Semantics

In this module I wrote and ran simple Java programs, getting comfortable 
with the basics of Java syntax and semantics.

## Topics covered

- Getting started with BlueJ (the Java IDE used throughout the course)
- Variables and mathematical operators
- Functions and conditionals
- Classes, types, and for-each loops

Most of this module consisted of videos, readings, and practice quizzes. 
The final graded coding assignment — calculating the perimeter of a shape — 
is included below.

## Project: Calculating the Perimeter of a Shape

[→ perimeter-assignment](./perimeter-assignment)

Given a set of points representing the vertices of a shape, this program 
calculates the shape's perimeter by summing the distances between 
consecutive points, and reports additional information such as the largest 
and smallest side lengths.
