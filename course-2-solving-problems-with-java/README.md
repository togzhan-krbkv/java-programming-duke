# Course 2: Java Programming: Solving Problems with Software

This course introduces core Java syntax and problem-solving skills, 
building up to a mini-project analyzing US baby name data across multiple CSV files.

## What I learned

- Edit, compile, and run Java programs
- Use conditionals, for loops, and while loops
- Use Java API documentation to work with unfamiliar classes and packages
- Debug programs systematically (the seven-step approach)
- Write methods to solve specific problems, and design test cases for them
- Combine multiple methods within a class to solve larger problems
- Apply divide-and-conquer techniques across multiple methods
- Work with strings: pattern matching, substrings, and multi-step text processing
- Parse and analyze CSV data using the Apache Commons CSV library, including across multiple files in a directory

## Development environment

This course uses BlueJ, a Java IDE developed by Duke University 
specifically for this specialization, along with a set of custom classes 
(`edu.duke.*`) such as `FileResource`, `DirectoryResource`, `URLResource`, 
and `StorageResource` that simplify file I/O and data access for 
beginners. All code in this course was written and tested in BlueJ.
