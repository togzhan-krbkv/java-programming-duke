# Course 3 Java Programming: Arrays, Lists, and Structured Data

Part of Duke University's Java Programming and Software Engineering 
Fundamentals Specialization on Coursera. This course builds on core Java 
skills with new data structures, object-oriented design, and more complex 
programs — including a cryptography project and a Vigenère cipher-breaking 
capstone.

## What I learned

- Reading and writing data from/to files
- Solving problems involving data files
- Quantitative analysis of data (maximums, minimums, averages)
- Storing and manipulating data in arrays and ArrayLists
- Combining multiple classes to solve larger problems
- Using iterables and collections, including HashMap and HashSet
- String concatenation and building strings with StringBuilder
- Refactoring programs for better organization using object-oriented principles

## Modules

- [ module-2-cryptography](./module-2-cryptography) — 
  Implementing and breaking classic ciphers (from Roman-era through early 
  1900s) using string processing and arrays
- [ module-3-gladlibs](./module-3-gladlibs) — 
  A randomly generated story generator using ArrayList and HashMap, 
  including a word frequency counter
- [ module-4-web-server-logs](./module-4-web-server-logs) — 
  Parsing web server logs to extract visitor IP addresses, timestamps, 
  unique visitor counts, and visit frequency
- [ module-5-miniproject-vigenere-cipher](./module-5-miniproject-vigenere-cipher) — 
  Capstone project: breaking the Vigenère Cipher in stages, using HashSet 
  and HashMap, expanding from known key length/language to unknown

## Note

Module 1 (Welcome) consisted only of an introductory video and readings
